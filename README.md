# 每日消息 Agent 网页版

这是一个“打开就自动更新”的每日新闻消息页。

## 功能

- 打开网页自动拉取最新新闻
- 每 10 分钟自动刷新一次
- 按重要性排序
- 分为：今日必看、科技AI、政治国际、A股经济、影视AI行业、生活风险
- 每条新闻给出：为什么重要、可能影响、你该怎么做
- 手机端适配，可添加到桌面作为 PWA

## 本地运行

```bash
npm install
npm run dev
```

打开：`http://localhost:3000`

## 部署到 Vercel

1. 把本项目上传到 GitHub
2. 打开 Vercel，选择 New Project
3. 导入这个 GitHub 仓库
4. Framework 选择 Next.js
5. 点击 Deploy

部署完成后，你会得到一个可以直接手机浏览器打开的网址。

## 数据源说明

当前版本默认使用公开新闻 RSS，无需 API Key。它适合个人使用和验证流程。

如果你要更稳定、更强的实时新闻数据源，可以后续接入 NewsAPI、GNews，或者国内新闻/财经数据接口。
